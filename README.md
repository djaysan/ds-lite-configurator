# DS Lite configurator

A single-file web configurator for custom Nintendo DS Lite shell colors. Pick colors for the shell, buttons, D-pad and hinge, or start from a preset theme, then download your design as a PNG.

## Features

- Recolor individual parts: top and bottom shell, hinge, D-pad, A/B/X/Y buttons, Start/Select
- Preset themes: SFC, SNES, DMG, Monster, GameCube, plus a reset option
- PNG export of your design (rendered with html2canvas)
- No build step, no backend, just one HTML file

## Usage

Open `index.html` in a browser, or host it on any static host (GitHub Pages, Cloudflare Pages, etc.).

Note: the page loads the Inter font and the html2canvas library from CDNs, so an internet connection is needed for full functionality.
